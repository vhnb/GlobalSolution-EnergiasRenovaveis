# GS-SOLUÇÕES EM ENERGIAS RENOVÁVEIS E SUSTENTÁVEIS

Sistema de monitoramento de missao espacial experimental com foco em energias renovaveis.
Desenvolvido para a Global Solution da FIAP - Ciencia da Computacao - SOLUÇÕES EM ENERGIAS RENOVÁVEIS E SUSTENTÁVEIS.

## o que faz

- simula dados de 4 modulos (Solar, Eolico, Comunicacao, Propulsao)
- monitora temperatura, energia, sinal e status em ciclos
- gera alertas quando algo sai dos limites
- toma decisoes automaticas com base nos alertas
- exibe resumo final com todos os alertas do ciclo

## como rodar

precisa ter Python 3 instalado. sem dependencias externas.

```
python main.py
```
